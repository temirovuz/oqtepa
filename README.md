## FastFood

___

* **Git clone project**
* **A virtual environment is created**
* **Installing requirements.txt**

        pip install -r requirements.txtv
* **makemigrations**

        python3 managa.py makemigrations
* **migrate**

        python3 managa.py migrate
* **project run**

        python3 managa.py runserver
